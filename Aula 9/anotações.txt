  -- Documento

Banco de dados é o conjunto de coleçoes (tabelas) que possui vários documentos e cada um tem seu respectivo campo

Diversos bancos de dados formam um cluster

Não existe interface gráfica no próprio banco de dados. Existem softwares de banco de dados e outro de interface gráfica

Evitar caracteres especiais - Não começar com número, não conter espaços, não utilizar palavras especiais

Não existem 2 documentos com o mesmo id
